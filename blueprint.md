
# Lotto Number Generator

## Overview

This is a simple web application that generates random lottery numbers.

## Current Plan

- **index.html**: Create the basic structure of the web page, including a title, a button to generate numbers, and a container to display the numbers.
- **style.css**: Style the page to be visually appealing, with a clean layout and modern design.
- **main.js**: Implement the logic for generating unique random lottery numbers and displaying them on the page.
